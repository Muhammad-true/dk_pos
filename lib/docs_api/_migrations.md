# Контракт таблицы `_migrations`

Служебная таблица: учёт применённых SQL-миграций. Не используется бизнес-логикой приложения.

## Движок

`InnoDB`, `utf8mb4`, `utf8mb4_unicode_ci`

## Колонки

| Колонка | Тип | NULL | По умолчанию | Описание |
|---------|-----|------|--------------|----------|
| `id` | `INT UNSIGNED` | NO | AUTO_INCREMENT | Первичный ключ |
| `name` | `VARCHAR(255)` | NO | — | Уникальное имя файла миграции |
| `executed_at` | `TIMESTAMP` | YES | `CURRENT_TIMESTAMP` | Время применения |

## Ключи

- **PRIMARY KEY:** `id`
- **UNIQUE:** `name`

## Внешние ключи

Нет.
